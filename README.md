# Guia para empacotamento
1. Baixe o conteúdo
2. Baixe [Build Tools for Visual Studio 2022](https://visualstudio.microsoft.com/downloads/)
3. Prepare o ambiente para uso de [Flutter](https://docs.flutter.dev/install)
4. No instalador escolha "Desktop development with C++".
5. Abra a pasta
6. Use o comando para empacotar: `flutter build windows --release`
7. Acesse a pasta gerada "build\windows\x64\runner\Release\"
